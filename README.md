# AgriBizProject Overview
AgriBiz is a responsive, multi‑page website designed to present an agribusiness company’s profile, services, products, and industry insights. The project demonstrates how modern web technologies can be applied to build a professional business presence online, with emphasis on usability, accessibility, and scalability.
The website is structured into five main pages:
Home – Introduction and mission statement 
Articles – Industry insights and agribusiness thought leadership
Products – Showcase of organic produce, equipment, and processed goods
Services – Consulting, investment advisory, logistics, and trade facilitation
Contact – Contact form and company details

Features
Responsive Design: Media queries ensure the site adapts seamlessly to desktops, tablets, and mobile devices.
Dropdown Navigation: Clean navigation bar with hamburger menu toggle for small screens.
Modern Typography: Uses Google Fonts (Poppins) with sans‑serif fallback for a professional look.
Contact Form: Simple form for inquiries, collaborations, or investment opportunities.

 Technologies Used

 HTML5 – Semantic structure for content
 CSS3 – Styling, layout, and responsive media queries
 JavaScript (ES6) – Navigation toggle for mobile view
 Google Fonts – Poppins for modern typography

Project Presentation Notes
Objective: Demonstrate a professional agribusiness website with responsive design.
Audience: Investors, partners, and clients interested in agribusiness solutions.
Key Value: Shows how technology can support agriculture by providing visibility, credibility, and accessibility online.
Future Enhancements: Integration with a CMS for dynamic articles, e‑commerce functionality for products, and backend support for contact form submissions.
